# Tarea: Optimización de Algoritmos y Complejidad Computacional
## Objetivo
El propósito de esta tarea es que los estudiantes analicen la eficiencia de los algoritmos proporcionados, identifiquen sus puntos críticos y propongan mejoras en rendimiento mediante la optimización del código.

## Parte 1: Optimización de Algoritmo con Listas y Condiciones
### Descripción
Tienes un algoritmo que procesa una lista de datos, realiza transformaciones y ejecuta verificaciones condicionales. Se ha identificado que su complejidad puede mejorarse.

### Tareas
Analiza la complejidad del algoritmo en su estado actual utilizando notación Big-O.
Identifica los elementos del código que pueden ser optimizados para reducir la complejidad.
Optimiza el código asegurando que:
Se reduzcan iteraciones innecesarias.
Se minimicen cálculos repetitivos.
Se mejore el uso de estructuras de datos más eficientes.
Compara el rendimiento del código original y el código optimizado, explicando las mejoras aplicadas.
## Parte 2: Optimización de Algoritmo con Recursión y Matriz
### Descripción
Este algoritmo manipula una matriz, ejecuta una recursión no optimizada y busca valores dentro de una lista. Se ha identificado que la recursión utilizada tiene complejidad exponencial (O(2ⁿ)), lo cual impacta negativamente en el rendimiento.

### Tareas
Analiza la complejidad del código en su estado actual y justifica el impacto del uso de recursión.
Optimiza la función recursiva utilizando memoization o programación dinámica para mejorar su rendimiento.
Propón una alternativa para reducir la complejidad de la manipulación de la matriz sin afectar la precisión del resultado.
Mide y compara el tiempo de ejecución del código original y el código optimizado, justificando las mejoras implementadas.
### Entregables
Código optimizado con comentarios explicando los cambios realizados.
Informe breve con:

Análisis de complejidad antes y después de la optimización.
Comparación de tiempos de ejecución con diferentes tamaños de entrada.
Justificación de las estrategias de optimización aplicadas.
